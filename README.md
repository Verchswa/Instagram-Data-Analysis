# 📊 Instagram Data Analysis using Python

## 📌 Project Overview

This project analyzes raw Instagram profile data using Python. The raw data is read from a text file, cleaned, converted into a structured JSON format, and analyzed to answer key questions about Instagram profiles.

This project demonstrates fundamental Python concepts used in data analysis such as file handling, string processing, data cleaning, JSON handling, dictionaries, sets, loops, and functions.

---

## 🎯 Objectives

The project answers the following questions:

- Who has the maximum number of posts?
- Who has the maximum number of followers?
- Who follows the maximum number of people?
- How many unique profile categories are present?

## 🚀 Features

- Reads raw Instagram profile data from a text file
- Cleans and preprocesses numerical values
- Converts K and M follower counts into integers
- Stores structured data in JSON format
- Finds:
  - Maximum Posts
  - Maximum Followers
  - Maximum Following
- Counts unique Instagram profile categories
